## 🚀 Instalação

### Pré-requisitos
- Um navegador web moderno (Chrome, Firefox, Safari, Edge)
- (Opcional) Um servidor HTTP local para melhor performance

### Opção 1: Abrir localmente
1. Clone o repositório:
```bash
git clone <URL_DO_REPOSITORIO>
cd portfolio
```

2. Abra o arquivo `index.html` diretamente no navegador

### Opção 2: Com servidor local (recomendado)
```bash
# Usando Python 3
python -m http.server 8000

# Usando Node.js (com http-server)
npx http-server
```

Acesse em `http://localhost:8000`

## 💡 Dicas de Personalização

1. **Alterar informações pessoais**: Edite o `index.html`
2. **Modificar cores**: Ajuste as variáveis CSS em `style.css`
3. **Adicionar/remover seções**: Edite o HTML e correspondente CSS
4. **Trocar imagens**: Substitua os arquivos em `/img`
5. **Adicionar vídeos**: Coloque os vídeos em `/videos` e atualize as referências
